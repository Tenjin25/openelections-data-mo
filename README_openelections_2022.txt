# Missouri 2022 County-Level Election Data

This branch adds:
- Data/20221108__mo__general__county.csv: County-level results for the 2022 Missouri general election (U.S. Senate, State Auditor)
- Data/2022_results.pdf: Original PDF from Missouri Secretary of State (117 pages)
- scripts/extract_2022_pdfplumber.py: Python script to extract tables from the PDF using pdfplumber
- scripts/consolidate_2022_data.py: Python script to consolidate extracted CSVs into a single county-level file

All files are provided for transparency, reproducibility, and OpenElections data integration.

For questions or methodology, see the scripts or contact the contributor.
